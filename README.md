# Hi, Anxiety

Недельный дневник приёма лекарств и самочувствия (сон, триггеры, спорт, события). Идея приложения — не бороться с тревогой, а знакомиться с ней: отмечать, понимать и тем самым прерывать замкнутый круг.

## Запуск

```bash
npm install
npx expo start
```

## Возможности

- Неделя с понедельника, листание недель и выбор месяца
- Строки лекарств: время (`HH:mm`), название, «принято», напоминание (колокол)
- Секции на каждый день: сон, триггеры, спорт, события
- Разворачивание длинного текста (стрелка вниз)
- 8 языков интерфейса
- Локальное сохранение; напоминания через `expo-notifications` (iOS/Android)

## Напоминания

1. Укажите время в формате `08:00` и название препарата.
2. Включите колокол в строке (по умолчанию включён).
3. В календаре нажмите кнопку уведомлений, чтобы выдать разрешение ОС.

Напоминания планируются на конкретные даты из дневника (будущие приёмы).
